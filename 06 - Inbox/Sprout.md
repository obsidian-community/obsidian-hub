# Sprout

Sprout is an Obsidian plugin for creating and reviewing flashcards directly within Markdown notes. It uses an FSRS-based scheduler and provides native tools for card creation, review, management and analysis inside Obsidian.

## Functionality

Sprout supports:
- Inline flashcard creation within Markdown notes 
- Spaced-repetition scheduling using FSRS 
- Reviewing cards without leaving Obsidian
- Local storage of card and scheduling data

## Features

- **FSRS scheduler** for spaced repetition
- **Card types**: basic, cloze, multiple choice and image occlusion
- **Inline editor** for creating and editing cards
- **Card browser** with search and filtering
- **Bulk-edit tools** for managing multiple cards
- **Reading-view cards** for reviewing excerpts or highlights without modifying note structure
- **Analytics dashboard** with charts and heatmaps

## Interoperability

- **Anki import/export (experimental)**, including scheduling data and media files
