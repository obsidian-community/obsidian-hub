## Features

- [[Themes with Friendly Settings|Friendly settings]]: Supports the [[obsidian-style-settings|Style Settings]] plugin

## Customization Options (Style Settings Plugin) 

**Colors**: 
- Accent Color: Choose the primary accent color for the theme.

**Style**: 
- Shadow Style: Controls the depth and weight of neo-brutalist shadows.
- Border Weight: Controls stroke weight on bordered elements.

**Typography**: 
- Heading Scale: Controls the relative size of h1–h4 headings.

**Features**: 
- Animations: Enable entrance animations and hover transitions throughout the UI.
