## Features

- [[Themes with Friendly Settings|Friendly settings]]: Supports the [[obsidian-style-settings|Style Settings]] plugin

## Customization Options (Style Settings Plugin) 
- Accent color: Primary accent used for links, cursor, active states. Terracotta orange by default (Claude palette).
- UI corner radius: Rounded-ness of buttons, inputs, callouts, cards.
- Body font: Font family for notes. Keep the quotes and commas.
- Monospace font: Font family for code and inline code.
- Base font size: Editor font size in pixels.
- Body line height: Line spacing for notes. 1.6-1.8 is comfortable for long reading.
- H1 weight: How bold H1 headings appear.
- Bold text weight: Weight applied to **bold** text.
