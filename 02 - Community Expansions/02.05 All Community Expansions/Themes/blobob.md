## Features

- [[Themes with Friendly Settings|Friendly settings]]: Supports the [[obsidian-style-settings|Style Settings]] plugin

## Customization Options (Style Settings Plugin) 

**Look & Feel**: 
- Dark Theme Colors: Set the theme colors for dark mode
- Light Theme Colors: Set the theme colors for light mode
- Layout & Tab Style: Set the workspace layout for desktop
- Bubble Layout - Color options: Only affects dark mode
- **Icons & Buttons**: 
    - Select, Buttons & Input Radiuses: Set the border radius for selects, buttons, and inputs
    - Button & dropdown select Style
    - Button Animation Style
    - Icon style: Set the icon style
- **UI Elements**: 
    - Focus mode: Removes UI elements. You should assign a hotkey to use it.
    - Inline Title Alignment
    - View Header title casing: Change the text case of the view header title
    - Hide Breadcrumbs
    - Ribbon style: Set the ribbon style
    - Status bar behaviour
    - Color Minimize & Maximize buttons
    - Hide Vertical Lines in Tables

**Editor & Text Appearance**: 
- Highlight styles: change the style of ==highlight==
- Lined Notebook: Adds lines to the editor
- **Active Line Behaviour**: 
    - Highlight active line
    - Highlight active line with face
    - Big active line text
    - Center active line
    - Darken inactive lines
    - Make inactive lines float
    - **Active Line Prefix**: 
        - Enable active line prefix: Text shown before the active line
        - Active line prefix text: Customize the text shown before the active line (e.g. ">>" or "→")
        - Active line prefix margin: Negative value pulls the prefix outside the editor edge
        - Active line prefix color: Color of the prefix text shown before the active line
- **Text Emphasis Colors**: 
    - Italic Text Color: Sets the color for all italic text
    - Bold Text Color: Sets the color for all bold text
    - Bold Italic Text Color: Sets the color for all bold italic text
- **Heading Colors**: 
    - All Headings Color: Set the color of all headings
    - Enable Per-Heading Colors: Allow individual color selection for each heading level (H1-H6)
    - H1 Color: Set the color for H1 headings
    - H2 Color: Set the color for H2 headings
    - H3 Color: Set the color for H3 headings
    - H4 Color: Set the color for H4 headings
    - H5 Color: Set the color for H5 headings
    - H6 Color: Set the color for H6 headings
- **Lists & Checkboxes**: 
    - Enable custom bullets
    - 1st level bullet
    - 2nd level bullet
    - 3rd level bullet
    - Disable checkboxes: Disable this if you don't want to use blobob theme's checkboxes.
    - Hide Strikethrough in checked list

**Tabs & Navigation**: 
- Hide tab list
- Hide tab close button
- Show only icon on pinned tabs
- Hide Left Sidebar Toggle Button
- Hide Right Sidebar Toggle Button
- Tab Style (Left top): Style for the top-left sidebar tab strip
- Tab Style (Right top): Style for the top-right sidebar tab strip
- Tab Style (Left): Style for the left sidebar tab strip
- Tab Style (Right): Style for the right sidebar tab strip

**File Explorer**: 
- Folders Style: Select the folder style and color accent
- Bouncy Folders: Adds a bounce animation to folders. Works with any folder style.
- Folder Border Radius: Border radius of folders
- Bold folder title: Make folder names bold in the file explorer
- Center folder title
- Hide folder Collapse Arrow: Hides the chevron/arrow icon on folder titles (but shows iconic icons ;D)
- File Explorer active file styling
- File Border Radius: Border radius of files
- Hide Action Buttons in File Explorer with Text: Text shown when buttons are hidden in file explorer. USE "" between your text
- File explorer indentation guides thickness

**Properties Panel**: 
- Properties Style: Select the properties layout style
- Dim Properties unless hovered
- Hide arrow in properties heading
- Center properties heading
- Cssclasses Style
- Properties metadata key Style
- Metadata property key casing: Change the text transform of property key labels

**Command Palette**: 
- Hide Instructions in Command Palette

**Animations**: 
- Disable settings item animation: Disables the hover lift animation on settings items
- Disable button, dropdown & input animations: Disables transition animations on buttons, dropdowns, and input boxes
- Disable clickable icon animations: Disables scale/transform animations on clickable icons
- Animation speed: Drag left for slower, right for faster animations. Default is the middle.

**Other Plugin Styling**: 
- Enable Kanban styling
- Kanban - take up full width: Makes kanban boards take up full width
- Kanban - Prepend Hover: Shows "+ Add a Card" button ON HOVER
- Kanban - cards increase size on hover: Increases font size ON HOVER
- Kanban - redden metadata date: Colors the date red
- Enable Image Captions styling
