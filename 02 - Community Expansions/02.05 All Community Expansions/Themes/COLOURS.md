## Customization Options (Style Settings Plugin) 

**Colours**: 
- Override System Accent: When off (default), Obsidian's appearance-settings accent is used. Turn on to use the per-mode colours below.
- Accent Colour: Only applied when 'Override System Accent' is enabled above.
- Heading Style: Monochrome uses a low-saturation accent tint (default). 'A little bit colour' and 'Go Crazy' progressively increase chroma and hue variation across heading levels.
- Heading Hue Rotate: Shift the base hue for 'A little bit colour' headings. 0 = accent hue. Positive values rotate warm, negative rotate cool.
- Coloured Alternate Checkboxes: Apply the Go Crazy hue palette to alternate checkbox backgrounds. Turn off to fall back to the Heading Style setting.
- Highlight Style: Colourful uses the accent complement hue (default). Other options reduce saturation or remove hue entirely.
- **Typography**: 
    - Heading Font: Set font for headings. Use quotes for names with spaces. 
 Default font: 'Overused Grotesk Roman', 'Helvetica Neue', 'Inter', sans-serif
    - Heading Sizes
    - Line Height
    - Enable Uppercase: Applies uppercase to the note title, table headers, and UI buttons
    - Font Brightness: Scales text opacity. Lower values give a softer, more muted look.
- **Layout**: 
    - Table Style
    - Folder Style
    - Internal Link Decoration
    - Auto-hide Scrollbar: Use the OS overlay scrollbar — appears while scrolling or hovering, hides otherwise. Replaces the theme's custom thin scrollbar.
    - Disable Cursor Line Highlight: Remove the background tint on the active editor line.
    - Hide Inline Title in Canvas: Show the inline note title in the editor but hide it inside Canvas cards.
    - Readable Line Width: Custom line width when Readable line length is on in Obsidian settings.
    - Base Embed Width Multiplier: Controls how wide embedded Bases are relative to the readable line width. 1 = same width, 5 = 5× wider.
