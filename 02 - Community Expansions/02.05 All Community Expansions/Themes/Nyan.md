## Features

- [[Themes with Friendly Settings|Friendly settings]]: Supports the [[obsidian-style-settings|Style Settings]] plugin

## Customization Options (Style Settings Plugin) 

**Fun Effects**: 
- Rainbow horizontal rules: Adds a rainbow gradient to horizontal rules (---)
- Rainbow headings: Each heading level gets a different pastel color
- Rainbow scrollbar: Adds a rainbow gradient to the scrollbar thumb
- Rainbow active tab indicator: Adds a rainbow gradient border to the active tab
- Pop-tart checkboxes: Checked boxes get a rainbow background

**Animations**: 
- Nyan Cat in status bar: Shows a pixel-art Nyan Cat with rainbow trail in the status bar
- Animated starfield background: Subtle twinkling stars on the editor background (dark mode only)
- Rainbow loading bar: Shows a rainbow progress bar at the top of active panes
- Sparkle on hover: Links and tags sparkle when you hover over them

**Color Customization**: 
- Accent color: Primary accent color used for links, buttons, and interactive elements
- Primary background: Main editor background color
- Secondary background: Sidebar and secondary panel background

**Typography**: 
- Editor font
- Base font size
