## Customization Options (Style Settings Plugin) 

**Palette Overrides**: Override individual palette colors.
- Foreground (fg): Main body text
- Foreground Bright (fg+1): Bold text
- Background (bg): Primary editor background
- Background Dark (bg-1): Deepest background, gutters, titlebar
- Background Light (bg+1): Secondary background, hover, active line
- Background Mid (bg+2): Borders, dividers
- Background Selection (bg+3)
- Background Match (bg+4): Search match, tooltips
- Yellow (keyword accent): Keywords, H1, active icons
- Green (strings): Strings, H4, success
- Niagara (links/functions): Links, functions, H2
- Wisteria (visited/H3)
- Brown (comments): Comments, H5, annotations
- Quartz (muted/constants)
- Niagara Dim (faint UI): Line numbers, faint text
- Red (errors)
- Red Bright (red+1)

**Editor**: 
- Base font size (px)
- Line height
- Max note width (ch): Maximum readable line width in characters
- Readable line length: Constrain note width to max-width
- Show line numbers
- Highlight active line
- Show indent guides
- Smooth scrolling

**Headings**: 
- H1 size (em)
- H2 size (em)
- H3 size (em)
- H4 size (em)
- Heading text transform

**Code Blocks**: 
- Code font size (em)
- Show border around code blocks
- Code block line numbers

**Interface**: 
- Sidebar width (px)
- Tab style
- Hide ribbon (left action icons)
- Compact status bar

**Callouts**: 
- Callout corner radius (px)
- Callout icon blend mode

**Vim & Cursor**: Cursor appearance and Vim mode settings. Enable Obsidian's built-in Vim keybindings under Settings > Editor > Vim key bindings.
- Cursor color (normal editing): Color of the text caret outside of Vim mode
- Vim NORMAL mode cursor shape
- Cursor blink / animation style
- Vim NORMAL mode cursor color
- Vim INSERT mode cursor color
- Vim VISUAL mode cursor color
- Vim REPLACE mode cursor color
- Vim mode indicator in status bar: Colorize the status bar based on current Vim mode (requires Vim Mode Status Bar or similar plugin)
