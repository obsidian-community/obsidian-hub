## Customization Options (Style Settings Plugin) 

**Synthwave Pro**: VaporWave / RetroWave theme — readable, low eye-strain. Toggle palette and effects below.
- Palette: Choose the color scheme. "Soft Reading" desaturates accents for long sessions.
- Background depth: How dark the editor background goes.
- Retro grid background: Subtle perspective grid behind the workspace. Off by default.
- Neon glow on headings & links: Adds a soft text-shadow. Off by default for max readability.
- CRT scanlines: Very subtle horizontal scanlines. Pure aesthetics — leave off if it bothers you.
- Monospace headings: Use a coding font for headings (programmer vibe).
- **Heading section**: 
    - Text font: Body text font family. Comma-separated list, no surrounding quotes.
    - Monospace / code font: Used for code blocks and (optionally) headings.
    - Base font size (px)
    - Line height
- **Custom accents**: 
    - Override primary accent: Leave empty to use palette default.
    - Override secondary accent
