## Customization Options (Style Settings Plugin) 

**Colors**: Theme Accent, Active Color, Background Intensity
- Primary Color: Main accent color (default: cyan)
- Secondary Color: Secondary color for hover and details (default: magenta)
- Active Color: Color for active/selected elements
- Background Intensity: How dark the background is

**Editor**: Bold, Italic, Headings styling
- Colored Bold: Bold text uses the primary theme color
- Colored Italic: Italic text uses the active theme color
- Neon Gradient Headings: Headings use neon gradient colors

**Workspace**: Layout, Cards, Tabs, Status Bar, Scrollbars, etc.
- Auto-hide Titlebar: Title bar hides until hover
- Slim Scrollbars: Minimalist scrollbars with accent color
- **Status Bar**: Select status bar style
    - Show Status Bar: Show the bottom status bar (hidden by default)
- **Tabs**: Tab bar appearance and behavior
    - Tab style
    - Tab Gap: Spacing between workspace tabs
    - Custom tab height: Adjust the height of workspace tabs
- **Workspace Layout**: Workspace layout and arrangement
    - Workspace Layout variant
    - Hide Borders: Remove UI borders for a cleaner look
    - **Card Layout Settings**: Card radius, padding, and component toggles
        - Card Layout — File Browser: Card style for the file explorer
        - Card Layout — Actions Panel: Card style for the side actions panel
        - Card Shadows: Add shadows to layout cards
