## Features

- [[Themes with Friendly Settings|Friendly settings]]: Supports the [[obsidian-style-settings|Style Settings]] plugin

## Customization Options (Style Settings Plugin) 

**Wallpaper**: 
- Wallpaper
- Solid Background: Replaces wallpaper with a flat color
- Background Color

**Colors**: 
- Primary: User text · headings · tags · active state bg
- Secondary: Internal interactive · nav · buttons
- Tertiary: External links · outgoing actions
- Caret
- Properties
- Blockquote
- **Headings**: 
    - H1
    - H2
    - H3
    - H4
    - H5
    - H6
- **Folders**: Top-level folders colored by position. Requires Colored Folders toggle in UI.
    - Folder 1
    - Folder 2
    - Folder 3
    - Folder 4
    - Folder 5
    - Folder 6
    - Folder 7
    - Folder 8

**UI**: 
- Underline Internal
- Underline External
- Hide Right Top Bar
- Hide Tab Headers
- Hide Ribbon
- Hide Vault Profile: Vault icon and switcher at the base of the left sidebar
- Hide Sidebar Toggle Buttons
- Hide File Explorer Buttons
- Hide System Icons: OS window controls
- Hide Status Bar
- Status Bar Background Color
- Hide Scrollbars
- Hide Tooltips
- Hide Prompt Instructions
- Hide Search Suggestions
- Hide Search Match Count
- Hide Properties in Reading View
- Maximize Media
- Image Grids
- Card Columns
- **Focus Mode**: 
    - Focus Mode: Collapses sidebars. Reveal on hover.
    - Reveal on Hover
