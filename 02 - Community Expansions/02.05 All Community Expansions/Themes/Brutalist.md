## Customization Options (Style Settings Plugin) 

**Typography & Typeface**: 
- Body Font: Main font for reading notes.
- Interface Font: Font for sidebar, tabs, and headers.

**Page Layout & Geometry**: 
- Note Width: Maximum width of the text block.
- Text Alignment: Reading alignment preference.
- Line Height: Vertical breathing room between lines.

**Dashboard & Masonry**: 
- Card Column Width: Minimum width of cards in masonry grid.
- Grid Gap: Space between cards.
