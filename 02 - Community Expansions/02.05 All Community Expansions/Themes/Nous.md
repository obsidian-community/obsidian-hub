- [[Themes with Friendly Settings|Friendly settings]]: Supports the [[obsidian-style-settings|Style Settings]] plugin

## Customization Options (Style Settings Plugin) 
- Headline font: The serif used for headings and display text.
- Body font: The serif used for body text in the editor and reading view.
- Small-caps font: The serif used for H5/H6 headings, callout titles, and table headers rendered in small-caps.
- UI font: The sans-serif used for the Obsidian interface (tabs, sidebars, menus).
- Monospace font: The monospace used for code, inline code, and front-matter values.
- Body text size: Base font size for editor and reading view body text.
- Body line height: Line height for body text. LaTeX-like generosity lives around 1.60–1.75.
- Reading width: Maximum width of the editor / reading column, in characters.
- Justify body paragraphs: LaTeX-style justified alignment with hyphenation for body text.
- Number headings (§1.1.1): Academic section numbering for H1 through H4 in reading view.
- Drop cap on first paragraph: LaTeX / editorial drop capital at the start of each note.
- Disable small-caps H5 / H6: Turn off the small-caps rendering for level-five and level-six headings.
