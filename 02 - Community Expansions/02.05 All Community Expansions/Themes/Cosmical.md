## Features

- [[Themes with Friendly Settings|Friendly settings]]: Supports the [[obsidian-style-settings|Style Settings]] plugin

## Customization Options (Style Settings Plugin) 

**Theme Colors**: 
- Subtheme: Choose a color subtheme that affects properties, tags, and accent color
- Use subtheme colors for headings: Applies subtheme colors to heading levels (h1-h6)
- Disable colored tags: Use neutral monochrome style for tags instead of subtheme colors (colored tags are enabled by default)
- Hollow tags style: Makes tags transparent with only an outline border

**Editor**: 
- Wider callouts: Makes callouts wider in the editor
- Custom caret color: Uses the subtheme color for the text cursor in the editor

**Headers**: 
- Headings font family: Font family used for note headers: inline-title, h1, h2, h3, h4, h5, h6
- Underline headings: Adds underline decoration to all heading levels
- Center file title: Centers the inline file title instead of left-aligned
- Hide file title in topbar: Completely hides the file title from the topbar
- Hide file path in topbar: Hides only the file path (parent folder) from the topbar, keeping the file name visible

**Sidebars**: 
- Hide right sidebar toggle button: Hides the right sidebar toggle button

**Properties settings (Metadata)**: 
- Options for the metadata (properties) container: Alternative visual styles for the properties container
- Enable bold for property labels: Makes property names display in bold font weight
- Show "Properties" label: Displays the "Properties" header above the properties section
- Improve multi-select list UX: Hides the remove button on multi-select pills until focused for a cleaner appearance

**Community Plugins**: 
- Colorful Bases tables: Applies subtheme colors to Bases plugin tables
