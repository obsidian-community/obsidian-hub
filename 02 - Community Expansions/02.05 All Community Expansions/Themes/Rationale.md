## Features

- [[Themes with Friendly Settings|Friendly settings]]: Supports the [[obsidian-style-settings|Style Settings]] plugin

## Customization Options (Style Settings Plugin) 

**Color Scheme**: 
- Color Scheme: Choose a preset color palette

**Custom Colors**: Override the color scheme with custom colors
- Custom Accent Color: Override accent color (leave empty to use scheme default)

**Typography**: 
- Body Font: Serif font for body text
- Heading Font: Sans-serif font for headings
- Monospace Font: Font for code blocks

**Layout**: 
- Border Radius: Roundness of UI elements
- Line Width: Maximum width of text lines (0 = unlimited)

**Tabs**: 
- Highlight Active Tab: Make the active tab slightly darker to stand out
- Active Tab Color: Background color for the active tab (when highlight is enabled)
- Active Tab Color (Dark Mode): Background color for the active tab in dark mode
- Hide Top Tab Bar (Vertical Tabs): Hide the top bar above the main editor when using vertical tabs (removes + button area)
- Tab Font: Font family for tab labels (vertical tabs)

**Text**: 
- Justified Text: Justify paragraph text in both reading and editing modes
- Floating Status Bar: Float the status bar in the center bottom of the screen

**File Explorer**: 
- Show File Explorer Icons: Display icons next to folders and markdown files in the file explorer
- Bold Folder Names: Make folder names bold in the file explorer
- Highlight Breadcrumb Path: Highlight ancestor folders of the active file in the sidebar
