# Copy Note Content

Plugin ID: `copy-as-note`
Links: [GitHub repository](https://github.com/lyqed/copy-as-note) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=copy-as-note)
Developed by: [[lyqed]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Copy notes to clipboard with title prepended. Supports single notes, whole folders, and a toggle to include/exclude YAML frontmatter.
