# Simple Folder Note

Plugin ID: `simple-folder-note`
Links: [GitHub repository](https://github.com/heono3h/simple-folder-note) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=simple-folder-note)
Developed by: [[heono3h]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Treats a .md file with the same name as its folder as a folder note. Hides it from the file explorer and opens it when the folder is clicked.
