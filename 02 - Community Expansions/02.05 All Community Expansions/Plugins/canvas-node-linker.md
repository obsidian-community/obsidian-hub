# Canvas Node Linker

Plugin ID: `canvas-node-linker`
Links: [GitHub repository](https://github.com/mvstro/canvas-node-linker) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=canvas-node-linker)

A simple plugin that adds a right-click menu on Canvas nodes to instantly copy a wikilink of the node.
