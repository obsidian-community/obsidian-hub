# CJK Cross-Script Search

Plugin ID: `cjk-cross-search`
Links: [GitHub repository](https://github.com/sai1047976/obsidian-cjk-cross-search) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=cjk-cross-search)
Developed by: [[sai1047976]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Type 萬維鋼 and find 万维钢 (and vice versa) in global search. Uses OpenCC to expand queries across Simplified and Traditional Chinese.
