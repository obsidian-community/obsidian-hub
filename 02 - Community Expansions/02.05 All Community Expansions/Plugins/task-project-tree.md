# Task Project Tree

Plugin ID: `task-project-tree`
Links: [GitHub repository](https://github.com/toustifer/obsidian-task-project-tree) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=task-project-tree)
Developed by: [[toustifer]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Turn folders into task projects with overview notes and tree-based progress. Right-click any folder to set it as a project, then see live task completion badges in the file explorer.
