# Style Tags in Search Results

Plugin ID: `style-tags-in-search-results`
Links: [GitHub repository](https://github.com/fanis/obsidian-style-tags-in-search-results) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=style-tags-in-search-results)
Developed by: [[fanis]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Detect hashtags in the Search pane and apply to them a configurable CSS class to enable styling. Optional one-click hide via injected CSS.
