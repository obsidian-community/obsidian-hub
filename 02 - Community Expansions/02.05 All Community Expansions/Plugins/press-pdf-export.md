# Press PDF Export

Plugin ID: `press-pdf-export`
Links: [GitHub repository](https://github.com/taylorchen/obsidian-press) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=press-pdf-export)
Developed by: [[taylorchen]]
Mobile compatible: [[Desktop-only plugins|No]]

High-fidelity PDF export powered by Pandoc. Supports multiple engines (XeLaTeX, wkhtmltopdf, WeasyPrint), Mermaid diagrams, custom CSS/templates, and batch export.
