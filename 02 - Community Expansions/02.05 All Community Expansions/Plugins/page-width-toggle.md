# Page Width Toggle

Plugin ID: `page-width-toggle`
Links: [GitHub repository](https://github.com/heono3h/page-width-toggle) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=page-width-toggle)
