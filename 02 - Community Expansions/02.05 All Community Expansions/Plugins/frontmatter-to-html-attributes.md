# Frontmatter to HTML Attributes

Plugin ID: `frontmatter-to-html-attributes`
Links: [GitHub repository](https://github.com/IllDepence/obsidian-frontmatter-to-html-attributes) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=frontmatter-to-html-attributes)
Developed by: [[IllDepence]]
