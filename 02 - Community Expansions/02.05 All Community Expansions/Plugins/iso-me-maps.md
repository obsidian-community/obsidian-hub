# iso.me Maps

Plugin ID: `iso-me-maps`
Links: [GitHub repository](https://github.com/codybontecou/isome-maps) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=iso-me-maps)
Developed by: [[codybontecou]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Render Leaflet maps inline from iso.me exports. Supports JSON, CSV, Markdown, OwnTracks, Overland, and GPX formats with stats and duration-encoded markers.
