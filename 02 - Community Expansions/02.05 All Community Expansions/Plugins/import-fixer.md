# Import Fixer

Plugin ID: `import-fixer`
Links: [GitHub repository](https://github.com/tonylee2016/obsidian-import-fixer) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=import-fixer)
Developed by: [[tonylee2016]]
Mobile compatible: [[Desktop-only plugins|No]]

Cleans up web-clipped notes (Evernote, Reddit, Zhihu, etc.): repairs broken resource links and shrinks oversized header favicons, avatars, and UI icons.
