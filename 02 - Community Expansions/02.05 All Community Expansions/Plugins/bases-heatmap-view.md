# Bases heatmap view

Plugin ID: `bases-heatmap-view`
Links: [GitHub repository](https://github.com/serafinovsky/bases-heatmap-view) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=bases-heatmap-view)
Developed by: [[serafinovsky]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Github-style heatmap visualization for any date-based property in your Bases tables. Track activity, habits, progress, or any numeric/date-related data with color-coded squares.
