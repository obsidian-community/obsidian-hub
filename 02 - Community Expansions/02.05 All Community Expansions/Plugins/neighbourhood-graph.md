# Neighbourhood Graph

Plugin ID: `neighbourhood-graph`
Links: [GitHub repository](https://github.com/agilesmagile/obsidian-neighbourhood-graph) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=neighbourhood-graph)
Developed by: [[agilesmagile]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Interactive D3.js force-directed graph showing the current note's neighbourhood in a sidebar panel. Nodes connected by shared tags and backlinks.
