# Tag Toggler

Plugin ID: `tag-toggler`
Links: [GitHub repository](https://github.com/studiogamma/tag-toggler) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=tag-toggler)
Developed by: [[studiogamma]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Hide specific tag nodes from Graph View by converting tags to plain text (e.g. #Year → —#Year). Unhide to restore them.
