# Custom Selectors

Plugin ID: `custom-selectors`
Links: [GitHub repository](https://github.com/hodie/obsidian-custom-selectors) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=custom-selectors)
Developed by: [[hodie]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Allows defining 'selector' properties (e.g., selector.status) which display as interactive dropdown menus with pre-defined options in both the standard Properties View and inside Bases views.
