# Scholar Sidekick — Cite from any identifier

Plugin ID: `scholar-sidekick`
Links: [GitHub repository](https://github.com/mlava/scholar-sidekick-obsidian) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=scholar-sidekick)
Developed by: [[mlava]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Paste a DOI, PMID, ISBN, or arXiv ID and replace it with a formatted citation. 10,000+ CSL styles. Check retraction and open-access status. Verify suspect citations. No reference manager required.
