# UniRate Currency

Plugin ID: `unirate-currency`
Links: [GitHub repository](https://github.com/unirate-api/obsidian-currency) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=unirate-currency)
Developed by: [[unirate-api]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Inline live and historical currency exchange rates. Type `usd:eur` in a note to see the rate; type `usd:eur:100` to convert. 170+ currencies and crypto via UniRateAPI.
