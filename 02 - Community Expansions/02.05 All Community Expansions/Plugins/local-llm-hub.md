# Local LLM Hub

Plugin ID: `local-llm-hub`
Links: [GitHub repository](https://github.com/takeshy/obsidian-local-llm-hub) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=local-llm-hub)
