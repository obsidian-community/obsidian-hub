# Vim Wikilink Suggest Navigation

Plugin ID: `vim-wikilink-nav`
Links: [GitHub repository](https://github.com/noxx/vim-wikilink-nav) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=vim-wikilink-nav)
Developed by: [[noxx]]
Mobile compatible: [[Desktop-only plugins|No]]

Vim-style Ctrl+N / Ctrl+P to navigate the inline wikilink suggestion ([[ ... ]]) only; Blocks global hotkeys only while popup is open. Desktop-only.
