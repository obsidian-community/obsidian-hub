# S3 Sync & Backup

Plugin ID: `simple-storage-sync-and-backup`
Links: [GitHub repository](https://github.com/ceilaolabs/obsidian-s3-sync-and-backup) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=simple-storage-sync-and-backup)
Developed by: [[ceilaolabs]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Vault synchronization and scheduled backups across devices using S3-compatible storage (AWS S3, Cloudflare R2, RustFS, etc.) with optional end-to-end encryption.
