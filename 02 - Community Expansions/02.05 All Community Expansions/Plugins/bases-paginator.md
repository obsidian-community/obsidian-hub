# Bases Paginator

Plugin ID: `bases-paginator`
Links: [GitHub repository](https://github.com/sean2077/obsidian-bases-paginator) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=bases-paginator)
Developed by: [[sean2077]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Adds a paginated table view with quick filtering to Bases. Features include pagination controls, global search, click-to-filter, and filter presets.
