# CSV Modern Codeblock

Plugin ID: `csv-modern-codeblock`
Links: [GitHub repository](https://github.com/vitovt/obsidian-csv-modern-codeblock) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=csv-modern-codeblock)
Developed by: [[vitovt]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Render and edit CSV and TSV code blocks as interactive tables with sticky headers, filters, sorting, layout toggles, link detection, and CSV validation.
