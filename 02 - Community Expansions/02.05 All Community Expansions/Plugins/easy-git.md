# Easy Git

Plugin ID: `easy-git`
Links: [GitHub repository](https://github.com/saiki77/Easy-Git) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=easy-git)
Developed by: [[saiki77]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

A secure, plug-and-play way to sync individual vault folders or sections with GitHub. Private repos supported, multiple folder mappings, bidirectional sync, and conflict resolution.
