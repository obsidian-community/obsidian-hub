# Recent Edits

Plugin ID: `recent-edits`
Links: [GitHub repository](https://github.com/cwagner223355/obsidian-recent-edits) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=recent-edits)
Developed by: [[cwagner223355]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Shows files modified in the last N days, grouped by day. Marks edits made from filesystem writes with a configurable indicator.
