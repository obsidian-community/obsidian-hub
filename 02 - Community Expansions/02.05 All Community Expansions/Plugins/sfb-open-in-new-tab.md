# Open in New Tab

Plugin ID: `sfb-open-in-new-tab`
Links: [GitHub repository](https://github.com/saltyfireball/obsidian-sfb-open-in-new-tab) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=sfb-open-in-new-tab)
Developed by: [[saltyfireball]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Forces files to open in new tabs instead of replacing the current tab. Covers all opening methods including Quick Switcher, command palette, bookmarks, search, backlinks, and graph view.
