# TTRPG Tools: Soundboard

Plugin ID: `ttrpg-soundboard`
Links: [GitHub repository](https://github.com/jareika/obsidian-ttrpg-soundboard) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=ttrpg-soundboard)
Developed by: [[jareika]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Soundboard for TTRPG sessions: image tiles (adjustable height), per-title loop/fades, Stop All, root folder + subfolders, plus Playlists (gold tiles with sequential playback).
