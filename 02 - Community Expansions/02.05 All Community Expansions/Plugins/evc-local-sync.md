# EVC Local Sync to AI Agent

Plugin ID: `evc-local-sync`
Links: [GitHub repository](https://github.com/entire-vc/evc-local-sync-plugin) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=evc-local-sync)
