# Header Backlinks

Plugin ID: `header-backlinks`
Links: [GitHub repository](https://github.com/rekby/obsidian-header-backlinks) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=header-backlinks)
Developed by: [[rekby]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Displays anchor icons next to headings targeted by internal links and lets you navigate to every note that references each heading.
