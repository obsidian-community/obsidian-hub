# Copy on Selection

Plugin ID: `copy-on-selection`
Links: [GitHub repository](https://github.com/cofuente/obsidian-copy-selection) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=copy-on-selection)
Developed by: [[cofuente]]
Mobile compatible: [[Desktop-only plugins|No]]

Automatically copies highlighted text to the clipboard when the selection changes, mirroring Linux primary-selection behavior.
