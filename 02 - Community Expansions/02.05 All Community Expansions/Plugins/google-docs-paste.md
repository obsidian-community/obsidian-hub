# Google Docs Paste

Plugin ID: `google-docs-paste`
Links: [GitHub repository](https://github.com/bfgpollara/obsidian-google-docs-paste) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=google-docs-paste)
Developed by: [[bfgpollara]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Paste from Google Docs with formatting (bold, italic, headings, lists, links, tables, images) preserved instead of collapsed to plain text.
