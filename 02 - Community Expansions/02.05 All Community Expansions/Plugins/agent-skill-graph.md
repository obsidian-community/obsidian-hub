# Agent Skill Graph

Plugin ID: `agent-skill-graph`
Links: [GitHub repository](https://github.com/hanamizuki/obsidian-skill-graph) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=agent-skill-graph)
