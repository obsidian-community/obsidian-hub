# Hidden Folders Access

Plugin ID: `hidden-folders-access`
Links: [GitHub repository](https://github.com/dsebastien/obsidian-hidden-folders-access) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=hidden-folders-access)
Developed by: [[dsebastien]]
Mobile compatible: [[Desktop-only plugins|No]]

Index hidden root-level folders (e.g. .claude) so they appear in the file tree, metadata cache, and Bases.
