# Local Graph Tag Links

Plugin ID: `local-graph-tag-links`
Links: [GitHub repository](https://github.com/lumyo/obsidian-local-graph-tag-links) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=local-graph-tag-links)
Developed by: [[lumyo]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Fixes the local graph to show all notes connected through shared tags. Notes attached to a tag are displayed even when the built-in local graph omits them.
