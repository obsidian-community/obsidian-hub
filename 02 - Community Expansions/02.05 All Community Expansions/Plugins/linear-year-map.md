# Linear Year Map

Plugin ID: `linear-year-map`
Links: [GitHub repository](https://github.com/michaelatdisney/ObsidianLinearYearMap) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=linear-year-map)
