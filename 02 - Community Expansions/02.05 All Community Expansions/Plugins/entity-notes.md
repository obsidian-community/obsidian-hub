# Entity Notes

Plugin ID: `entity-notes`
Links: [GitHub repository](https://github.com/bartvw/entity-notes) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=entity-notes)
Developed by: [[bartvw]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Create entity notes on the fly: write a dangling [[wikilink]], add a tag like #person or #project, and click the inline button to spin up a structured note. Links to entity notes are decorated with a colored pill in the editor.
