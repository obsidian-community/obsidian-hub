# Voxtral Transcribe

Plugin ID: `voxtral-transcribe`
Links: [GitHub repository](https://github.com/maxonamission/obsidian-voxtral) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=voxtral-transcribe)
Developed by: [[maxonamission]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Real-time streaming transcription using Mistral Voxtral, supports voice commands (headings, lists, to-dos and more) and automatic text correction.
