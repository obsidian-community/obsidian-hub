# Vault Sync (REST)

Plugin ID: `vault-sync-rest`
Links: [GitHub repository](https://github.com/andrewboldi/obsidian-vault-sync) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=vault-sync-rest)
Developed by: [[andrewboldi]]
Mobile compatible: [[Mobile-compatible plugins|Yes]]

Two-way sync with a GitHub repo via REST API. Works on iOS for vaults of any size including images, where git-protocol plugins crash from WebView memory limits.
