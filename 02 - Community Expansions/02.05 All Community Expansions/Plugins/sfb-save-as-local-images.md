# Save as Local Images

Plugin ID: `sfb-save-as-local-images`
Links: [GitHub repository](https://github.com/saltyfireball/obsidian-save-as-local-images) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=sfb-save-as-local-images)
