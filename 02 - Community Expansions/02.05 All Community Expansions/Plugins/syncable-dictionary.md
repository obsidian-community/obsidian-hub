# Syncable Dictionary

Plugin ID: `syncable-dictionary`
Links: [GitHub repository](https://github.com/masoniis/obsidian-syncable-dictionary) or [<button id=HH>Open in Obsidian</button>](obsidian://show-plugin?id=syncable-dictionary)
Developed by: [[masoniis]]
Mobile compatible: [[Desktop-only plugins|No]]

Maintains a global list of dictionary words and automatically updates the official spellcheck dictionary to match. This enables syncing of the official dictionary across desktop devices.
